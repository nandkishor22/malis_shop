## 🚀 Getting Started

### 📋 Prerequisites
- Node.js 18+
- npm 9+

### ⚙️ Installation
```bash
git clone https://github.com/nandkishor22/malis_shop.git
cd malis-shop
npm install
```

### 🏃 Usage
```bash
npm start
```
